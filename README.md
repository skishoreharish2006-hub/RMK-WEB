# RMK Engineering College - Modern React Website

A fully responsive, modern engineering college website built with React, Tailwind CSS, and Framer Motion. Inspired by the RMK Engineering College design but upgraded with premium UI/UX, animations, and advanced features.

## 🚀 Features
- **Modern UI/UX**: Premium aesthetic with glassmorphism, smooth gradients, and curated typography.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- **Dark Mode**: Integrated dark mode toggle with system persistence.
- **Advanced Animations**: Powered by Framer Motion and AOS (Animate on Scroll).
- **Interactive Pages**:
  - **Home**: Dynamic hero, stats counter, and featured departments.
  - **About**: College legacy and vision/mission.
  - **Departments**: Filterable list of academic departments.
  - **Faculty**: Searchable faculty directory.
  - **Admissions**: Detailed procedure and enquiry form with validation.
  - **Gallery**: Masonry layout with a full-screen lightbox modal.
  - **Contact**: Multi-field contact form with real-time validation.
- **Sticky WhatsApp**: Floating chat button for quick enquiries.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/) (Vite)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [AOS](https://michalsnik.github.io/aos/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **Forms**: [React Hook Form](https://react-hook-form.com/)

## 📂 Project Structure
```text
/src
  /components
    /layout     - Header, Footer, Layout Wrapper
    /ui         - WhatsAppButton
  /data         - JSON Mock Data (Departments, Faculty, etc.)
  /pages        - Home, About, Departments, Admissions, etc.
  /styles       - Tailwind & Global CSS
  App.jsx       - Routing & Animation Init
  main.jsx      - Entry Point
```

## 🛠️ Setup & Running Locally

1. **Clone/Download** the project folder.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Build for Production**:
   ```bash
   npm run build
   ```
